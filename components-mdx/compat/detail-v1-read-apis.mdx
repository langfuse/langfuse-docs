Deprecated. Removed on Langfuse v4 (Langfuse Cloud: removal date will follow). The full list and replacements:

| Deprecated `GET` endpoints                                                                                         | Replacement                                                                                                                   |
| ------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| `/api/public/observations`<br />`/api/public/observations/{id}`                                                    | `GET /api/public/v2/observations?fromStartTime={datetime}&toStartTime={datetime}`                                             |
| `/api/public/traces`<br />`/api/public/traces/{id}`                                                                | `GET /api/public/v2/observations?fromStartTime={datetime}&toStartTime={datetime}`<br />add `&traceId={id}` for a single trace |
| `/api/public/sessions`<br />`/api/public/sessions/{id}`                                                            | `GET /api/public/v2/observations?filter={urlencoded sessionId filter}&fromStartTime={datetime}&toStartTime={datetime}`        |
| `/api/public/scores`<br />`/api/public/scores/{id}`<br />`/api/public/v2/scores`<br />`/api/public/v2/scores/{id}` | `GET /api/public/v3/scores`                                                                                                   |
| `/api/public/metrics`<br />`/api/public/metrics/daily`                                                             | `GET /api/public/v2/metrics?query={json}`                                                                                     |

**Read APIs**

Move reads from the deprecated endpoints (`api.legacy.*` or direct REST calls) to the [Observations API v2](/docs/api-and-data-platform/features/observations-api#v2) and [Metrics API v2](/docs/metrics/features/metrics-api#v2). The [migration guide](/faq/all/deprecated-api-migration) has the per-endpoint replacements, parameter mappings, and before/after examples.

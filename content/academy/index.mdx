# Langfuse Academy

Welcome to the Langfuse academy! Here you'll learn about important concepts and best practices in LLM engineering. It's less about "how do I set up the integration in code" and more about "how do I design the setup and process to monitor and improve my AI agent effectively". You'll find conceptual guides, examples, .. for you to learn from. 

The following topics are discussed here:

- [LLM Observability](/academy/observability-overview) — what observability means for LLM apps, core concepts, and what good traces look like
- [Structured Evaluation](/academy/evaluation-overview) — the evaluation loop, designing evals, building datasets, and example setups
- [Prompt Management](/academy/prompt-management-overview) — when it makes sense, version control, deployment strategies, and composability
- [Monitoring & Improving in Production](/academy/monitoring-overview) — what to monitor, cost tracking, user feedback, and closing the loop

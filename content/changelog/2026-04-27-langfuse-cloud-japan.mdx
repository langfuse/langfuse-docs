Langfuse Cloud is now live in a dedicated Japan region, hosted in Tokyo (`ap-northeast-1`). Traces, prompts, and evaluation data stay in Japan. Langfuse Cloud Japan is available via https://jp.cloud.langfuse.com - you can sign up now.


## What is Langfuse Cloud Japan?

- **Full platform parity** with Cloud EU and Cloud US — same features, same API surface, same SDK support
- **Data stays in Japan** — traces, prompts, and evaluation records are stored and processed in Tokyo
- **Japanese-speaking team** based in Tokyo for sales and support
- **Migration path** from Cloud EU or Cloud US — [data migration script](/guides/cookbook/example_data_migration). You can reach out to us if you have a very large amount of trace data to migrate (>2M observations)

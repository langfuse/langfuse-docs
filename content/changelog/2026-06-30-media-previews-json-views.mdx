Langfuse now renders previewable media values as compact tags in JSON and table views. Langfuse media references, data URIs, and URLs with clear media extensions are detected automatically, so you can open the tag to inspect the underlying image, audio, or file without leaving the current trace, observation, or dataset view.

See the [multi-modality docs](/docs/observability/features/multi-modality#media-reference) for how Langfuse stores and resolves media references.

# Why do I see empty files in my blob storage export?

The [blob storage integration](/docs/api-and-data-platform/features/export-to-blob-storage) exports one time window of data per run, writing a file per table (for example `observations_v2/` and `scores/`). If you find empty files in your bucket, this is expected behavior — here's why, and how to handle it.

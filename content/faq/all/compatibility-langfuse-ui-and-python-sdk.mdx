# What is the compatibility between Langfuse application / UI and Python SDK?

Generally, the Langfuse application is backwards compatible with the current and the previous SDK major version of each language. When upgrading to a new SDK major version on a self-hosted deployment, make sure your server meets the SDK's minimum server version — not necessarily the latest release.

For the full SDK ↔ server compatibility matrix, minimum version requirements, and API version availability, see [Versions & Compatibility](/docs/compatibility).

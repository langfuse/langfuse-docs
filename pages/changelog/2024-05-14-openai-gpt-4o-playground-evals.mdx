OpenAI has delighted many developers with the [release of GPT-4o](https://openai.com/index/hello-gpt-4o/). The new model comes at improved performance, lower costs, greater speed, and more generous rate limits.

We see many users in the Langfuse community excited about the new model so we are happy to announce that we have added day-1 support for GPT-4o in the [Langfuse LLM playground](/docs/playground) and [evaluations](/docs/scores/model-based-evals). We already support [cost and usage tracking for the GPT-4o](/changelog/2024-05-14-openai-gpt-4o).

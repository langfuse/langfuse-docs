## New Features

- Introduced the complete OSS LLMOps Stack that integrates the robust LiteLLM unified API for seamless access to over 100+ LLMs.
- Enhanced observability and evaluation through Langfuse, providing detailed monitoring and prompt management capabilities.

## Improvements

- Streamlined integration between LiteLLM and Langfuse to ensure scalable, flexible, and reliable LLM operations.
- Optimized failover mechanisms and improved logging to elevate the user experience and system robustness.

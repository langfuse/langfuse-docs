If you [self-host](/self-hosting) Langfuse, [upgrade](/self-hosting/upgrade) your instance to >3.65 to take advantage of the new features.

If you use Langfuse Cloud, this change does not affect you.

## Changes

| Feature                                                      | `< v3.65.0` | `>= v3.65.0` |
| ------------------------------------------------------------ | ----------- | ------------ |
| Core Platform                                                | OSS (MIT)   | OSS (MIT) ✅ |
| [LLM-as-a-Judge Evaluations](/docs/scores/model-based-evals) | Commercial  | OSS (MIT) ✅ |
| [Playground](/docs/playground)                               | Commercial  | OSS (MIT) ✅ |
| [Prompt Experiments](/docs/datasets/prompt-experiments)      | Commercial  | OSS (MIT) ✅ |
| [Annotation/Data Labeling](/docs/scores/annotation)          | Commercial  | OSS (MIT) ✅ |
| [S3 Exports](/self-hosting/infrastructure/blobstorage)       | Commercial  | OSS (MIT) ✅ |
| [PostHog Integration](/integrations/analytics/posthog)       | Commercial  | OSS (MIT) ✅ |

Find more information on [our blog](/blog/2025-06-04-open-sourcing-langfuse-product). Please leave us any feedback via [GitHub Discussions](/ideas).

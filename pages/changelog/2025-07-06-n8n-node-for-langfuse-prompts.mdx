> [n8n](https://github.com/n8n-io/n8n) is a fair-code licensed workflow automation platform.

We're excited to announce the availability of a community-maintained n8n node for Langfuse! This integration brings Langfuse's prompt management capabilities directly into your n8n workflows.

## What's New

The Langfuse n8n node provides seamless integration with [Langfuse's Prompt Management](/docs/prompt-management/get-started) in n8n workflows. Key features include:

- **Fetch Prompts**: Retrieve prompts from your Langfuse project with support for versioning and labels
- **Compile Templates**: Process prompt templates with variables to generate ready-to-use prompts
- **Version Control**: Access specific prompt labels like `production` and `latest`

## Installation

Self-hosted n8n: Install via **Settings** > **Community Nodes** using package name: [`@langfuse/n8n-nodes-langfuse`](https://www.npmjs.com/package/@langfuse/n8n-nodes-langfuse)

n8n Cloud: Use the node directly in your workflows by searching for `Langfuse`.

# Langfuse Cloud

import { Steps } from "nextra-theme-docs";

## Overview

Langfuse cloud is a managed version of the langfuse server.

The langfuse core team takes care of:

- Deployment
- Scaling
- Upgrades and security patches
- Availability

## Get started

<Steps>

### Create an account

[Sign up](https://cloud.langfuse.com/auth/sign-up) for langfuse cloud.

### Create a project to get your API keys

### Configure SDKs to use the cloud environment

#### Server

```typescript
import { LangfuseClient } from "@finto-fern/langfuse-node";

const client = new LangfuseClient({
  environment: "https://cloud.langfuse.com",
  username: env.LANGFUSE_PUBLIC_KEY,
  password: env.LANGFUSE_SECRET_KEY,
});
```

#### Frontend

```typescript
import { LangfuseClient } from "@langfuse-fern/react-client";

const client = new LangfuseClient({
  environment: "https://cloud.langfuse.com",
  token: env.LANGFUSE_PUBLIC_KEY,
});
```

### Monitor your events

Go to [cloud.langfuse.com](https://cloud.langfuse.com) to monitor your events.

</Steps>

## Pricing

### Free-tier

The free-tier is a great way to get started with langfuse. It includes:

- 50,000 tracked events per month
- 30 days of history

### Usage-based pricing

Above the free-tier, pricing is based on the number of tracked events per month.

| Tracked events per month | Price              |
| ------------------------ | ------------------ |
| 50,000                   | $0                 |
| 50,000 - 100,000,000     | $1 / 10,000 events |
| 100,000,000+             | Contact us         |

If you have any other requirements, please contact us at: enterprise@langfuse.com.

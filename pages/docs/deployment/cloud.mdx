# Langfuse Cloud

import { Steps } from "nextra-theme-docs";

## Overview

Langfuse cloud is a managed version of the Langfuse server.

The Langfuse core team takes care of:

- Deployment
- Scaling
- Upgrades and security patches
- Availability
- [Data security and privacy](/docs/data-security-privacy)

All Langfuse Cloud plans include:

- Support via Slack, Discord and Email

## Get started

<Steps>

### Create an account

[Sign up](https://cloud.langfuse.com/auth/sign-up) for Langfuse cloud without credit card.

### Create a project to get your API keys

### Integrate with Langfuse

SDKs use Langfuse cloud as default baseUrl. No configuration is required.

### Monitor your events

Go to [cloud.langfuse.com](https://cloud.langfuse.com) to monitor your events.

</Steps>

## Pricing

### Free-tier

The free-tier is a great way to get started with Langfuse. It includes: 50,000 tracked events per month

### Usage-based pricing (ingestion)

Above the free-tier, pricing is based on the number of tracked events per month.

| Tracked events per month | Price              |
| ------------------------ | ------------------ |
| 0 - 50,000               | $0                 |
| 50,000 - 100,000,000     | $1 / 10,000 events |
| 100,000,000+             | Contact us         |

If you have any other requirements, please contact us at: enterprise@langfuse.com.

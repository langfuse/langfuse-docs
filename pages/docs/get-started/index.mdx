## What You'll Need

Before starting any of the quickstarts, you'll need:

1. A [Langfuse account](https://cloud.langfuse.com/auth/sign-up) (or [self-hosted instance](/self-hosting))
2. API credentials from your project settings
3. Your favorite programming language (we support Python, JavaScript/TypeScript, and many integrations)

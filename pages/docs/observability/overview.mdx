# What is Tracing?

Tracing captures everything happening during an LLM interaction: inputs, outputs, tool usage, retries, latencies and costs. Itâ€™s all tied to a unique trace ID and visualized step-by-step.

## What tracing can do for you

- Faster debugging 
- Cost tracking for each model call
- Performance tuning by spotting latency bottlenecks
- Saving hours in customer support
- Setting foundation for evaluations

# Metrics

If you [liked your prompt with Langfuse Tracing](/docs/prompt-management/features/link-tracing), you can track metrics for your prompt versions to see how they perform over time across different metrics or [scores](/docs/evaluation/data-model). 

<Callout type="info" emoji="ðŸ’¡">
For a structured analysis of different prompt versions you can run a [Prompt Experiment](/docs/evaluation/features/prompt-experiments) on a Langfuse Dataset.
</Callout>

To access the metrics, navigate to your prompt and click on the `Metrics` tab.

## Metrics Reference

- Median generation latency
- Median generation input tokens
- Median generation output tokens
- Median generation costs
- Generation count
- Median [score](/docs/evaluation/data-model) value
- First and last generation timestamp

## 🚧 In progress

- Webhooks for tracing, dataset, and admin events ([#1033](https://github.com/orgs/langfuse/discussions/1033))
- Unified agent graphs
- Langfuse JS OpenTelemetry SDK
- UI quality‑of‑life enhancements

## 🔮 Planned

- Agent Observability
  - Unified agent graphs (beta for LangGraph) 
  - New span type for tool calls
  - Agent evaluations: conversation, response, turn, and trajectory levels

- Evaluation
  - Rule‑based evaluators (regex, structural checks)
  - Trace LLM‑as‑judge evaluations for debugging & cost tracking
  - Evaluation comparison dashboard: correlation, confusion matrix, overlap histogram
  - SDK abstraction for easy experiment setup and UI‑triggered external runners
  - Sessions & observations support for annotation queues

- Datasets
  - Multi‑modal dataset support
  - Run comparison view: filterable, sortable dataset runs with annotation support
  - Bulk add traces to datasets
  - Dataset folders
  - Input/output schema validation

- Playground
  - Multi‑modal message support
  - Dataset experiments in playground

- Prompt Management
  - Store prompt variables in observation spans
  - Prompt experiments with structured outputs and tool/function support
  - Exports & imports for prompts

- Data Platform
  - Webhooks for tracing, dataset, and admin events
  - In-app routing (e.g. add to annotation queue, trigger evals)
  - Metric‑threshold alerts
  - Langfuse Cloud usage alerts

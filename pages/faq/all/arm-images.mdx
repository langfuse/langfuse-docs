# Are there prebuilt ARM images available?

Yes, since version [2.58.0](https://github.com/langfuse/langfuse/releases/tag/v2.58.0), there are prebuilt ARM images (`linux/arm64`) available. Thanks again [@rennokki](https://github.com/rennokki) for the contribution.
